# Conclusión final y plan de instalación

## 1. Resumen del análisis
Resume aquí qué habéis observado tras comparar y probar las 3 ISOs.

## 2. Decisión final
- **ISO elegida como opción principal:**MX
Rapida ligera y facil para el usuario.  
- **ISO elegida como alternativa:** antiX
La mas ligera para poderse instalar en cualquier sistema  
- **ISO elegida como respaldo:** Antix en rufus y Mint  

## 3. Justificación de la decisión
Tiene sentido porque de por si el ordenador tiene 4 modulos para memoria ddr2 aunque no estan instalados los modulos originales el ordenador sigue teniendo 3GB si se pudiera conseguir que subiera a 4GB se podrian istalar otros sistemas operativos como Windows 10 o Windows 11 pero sin TPM no merece la pena porque es mas dificil y pierde seguridad.
## 4. Plan para el aula taller
1º Entrar en la BIOS y ver si se puede poner la BIOS en modo UEFI
2º Poner el primero en orden de arranque el USB Booteable
3º Iniciar la iso de instalacion de MX
4º Si falla probar a instalar antiX
5º Si a fallado antix o no se puede por culpa de que el sistema no funciona con ventoy usar un USB booteable con rufus que tiene ya instalado antiX
6º Si ninguna iso a dado resultado intentar instalar mint este deberia ir el problema esque ira a baja velocidad por lo que intentare ver si se puede cambiar la RAM por otra compatible
## 5. Aprendizaje obtenido
Intentar adecuarnos a unos limites establecidos y intentar conseguirlo de la mejor forma posible
