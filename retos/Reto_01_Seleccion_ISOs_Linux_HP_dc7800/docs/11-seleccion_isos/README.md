# 11 · Selección de ISOs

En esta carpeta se realiza la selección de las **3 ISOs Linux** que se van a probar.

## Archivos de trabajo

- `fichas/tabla_comparativa.md`
- `fichas/iso_01.md`
- `fichas/iso_02.md`
- `fichas/iso_03.md`

## Requisitos mínimos del bloque

- Debes seleccionar **3 ISOs como mínimo**.
- Cada elección debe estar **justificada**.
- Debes indicar qué ISO será:
  - **opción principal**
  - **alternativa**
  - **respaldo**
- Debes citar **fuentes oficiales**.

## Consejo

No te centres solo en “que sea ligera”.  
También importa que la ISO sea descargable, que tenga instalador usable y que parezca razonable para el equipo.
