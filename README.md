# FHW · Proyecto RA3 · UT5

## Descripción general

Este repositorio contiene los retos del **Proyecto de FHW** para el **RA3** dentro de la **UT5: Respaldo y creación de imágenes de un sistema**.

La idea del proyecto es sencilla: antes de tocar un equipo antiguo de verdad, el alumnado debe **analizar**, **elegir**, **probar** y **documentar**.  
Igual que un técnico no llega con una sola herramienta al taller, aquí no se trabaja con una única ISO “porque sí”, sino con varias opciones justificadas.

## Estructura del proyecto

```text
retos/
├── Reto_01_Seleccion_ISOs_Linux_HP_dc7800/
├── Reto_02_Instalacion_en_HP_dc7800/
└── Reto_03_Comprobacion_postinstalacion/
```

## Flujo recomendado

1. Completar el **Reto 01**: selección de 3 ISOs ligeras y prueba de las 3 en máquina virtual.
2. Completar el **Reto 02**: instalación de la opción elegida en el equipo real.
3. Completar el **Reto 03**: comprobación, validación y cierre.

## Entrega

En cada reto el alumnado irá construyendo la documentación en Markdown.  
La entrega final de cada reto será **un único PDF** generado a partir del documento consolidado de la carpeta `docs/`.

## Recomendaciones para el alumnado

- Trabaja siempre con **fuentes oficiales**.
- Usa nombres claros para las capturas.
- No dejes las conclusiones para el final sin anotar incidencias durante el proceso.
- Piensa en este proyecto como una **maleta de viaje técnica**: si una ISO falla, debe existir otra preparada.

## Nota para el profesorado

Este repositorio está preparado como plantilla para ser **clonado o duplicado** por el alumnado y completado con su propia información, capturas y conclusiones.
