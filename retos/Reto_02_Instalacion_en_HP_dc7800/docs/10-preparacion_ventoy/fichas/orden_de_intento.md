# Ficha · Orden de intento de instalación

## Orden previsto

1. **Primera opción:**Antix
   - Motivo:Bajos recursos
2. **Segunda opción:**Mynt
   - Motivo:Altos recursos comparado a la anterior queriamos probar si el sistema daba la talla con un sistema con mas requisitos
3. **Tercera opción:**puppy
   - Motivo:Bajos recursos y diferente a antix. 

## Justificación breve

Tiene sentido probar Antix ya que es la que menos recursos pide, por eso al probar que funcionaba decidimos porbar otras isos y nos decidimos a probar mynt por que es la que mas recursos pedia de todas nuestras isos. Se probo un puppy porque aunque llevavamos un MX se probo el puppy porque si funcionaba Antix MX que son sistemas hermanos deberia ir.

## Regla de cambio

Indica cuándo decidiréis pasar de una ISO a la siguiente.  
Ejemplos:
- Al iniciar la iso con ventoy se queda trabado.
- El sistema se traba en el modo prueba del sistema operativo.
- El sistema no funciona al iniciar el grub.
