Enunciado.

EJERCICIO 1. Preparar un USB de instalación con Ventoy y copiar en él las 3 imágenes ISO seleccionadas en el Reto 01. Debe quedar documentado el proceso de preparación del USB y debe demostrarse con capturas que las tres ISOs están disponibles para arrancar.

EJERCICIO 2. Arrancar el HP Compaq dc7800 desde el USB con Ventoy e intentar instalar Linux en el equipo real. Se instalará al menos una ISO, pero las otras dos deben estar preparadas por si aparece algún fallo en la primera opción.

EJERCICIO 3. Documentar de forma completa todo el proceso: orden de intento de las ISOs, pasos principales de la instalación, capturas, incidencias encontradas, soluciones aplicadas y resultado final, dejando constancia de que el sistema ha quedado instalado.

Criterios de puntuación. Total 10 puntos.

EJERCICIO 1. 2 puntos

EJERCICIO 2. 5 puntos

EJERCICIO 3. 3 puntos

Recursos necesarios para realizar la Tarea.

Ordenador para preparar el USB, pendrive, programa Ventoy, las 3 ISOs Linux seleccionadas en el reto anterior, el equipo real HP Compaq dc7800, cámara o herramienta de capturas y un editor Markdown o procesador de texto para documentar el trabajo.

Consejos y recomendaciones.

Haz las fotografías o capturas durante el proceso y no al final. Anota enseguida cada error y la solución probada. Si la primera ISO falla, no improvises: pasa a la alternativa prevista y deja constancia de por qué has cambiado.

Recuerda que Ventoy es el medio obligatorio de instalación en este reto, por lo que el USB debe estar bien preparado y contener las tres opciones.

Indicaciones de entrega.

Una vez realizada la tarea elaborarás un único documento en PDF donde figuren las respuestas correspondientes. El envío se realizará a través de la plataforma de la forma establecida para ello, y el archivo se nombrará siguiendo el siguiente patrón: apellido1_apellido2_nombre_TareaXX

Asegúrate que el nombre no contenga la letra ñ, tildes ni caracteres especiales extraños. Así por ejemplo la alumna Begoña Sánchez Mañas para la octava tarea, debería nombrar esta tarea como... sanchez_manas_begona_Tarea08
