# Reto 01 · Selección de ISOs Linux ligeras para HP Compaq dc7800

## Objetivo del reto

Analizar las características del **HP Compaq dc7800** estudiado en clase, seleccionar **al menos 3 imágenes ISO** de distribuciones Linux ligeras adecuadas para ese equipo y **probar las 3 en una máquina virtual** con capturas de pantalla.

## Idea técnica del reto

Antes de instalar en un equipo antiguo real, conviene hacer una especie de **ensayo general**.  
La máquina virtual hace aquí el papel de una maqueta: no es idéntica al equipo real, pero permite comprobar si la ISO arranca, si el instalador es usable y si la instalación parece viable.

## Qué debe quedar demostrado

- Que el grupo conoce las características del equipo real.
- Que la selección de ISOs está **justificada**.
- Que se han probado **las 3 ISOs** en una VM.
- Que existen **capturas** del proceso.
- Que el grupo propone un orden razonable de instalación en el equipo real.

## Estructura interna del reto

- `assets/`: imágenes y capturas.
- `docs/`: documentación de trabajo y documento final consolidado.
- `rubric/`: rúbrica de evaluación.

## Producto final

El alumnado deberá generar un único PDF a partir de:

- `docs/90-ENTREGA_UNICA.md`

## Nombre sugerido del PDF

`apellido1_apellido2_nombre_TareaXX.pdf`

## Mínimos obligatorios

- 3 ISOs como mínimo.
- 3 fichas de ISO completas.
- 3 pruebas de VM completas.
- 3 capturas por ISO como mínimo:
  - arranque
  - instalador
  - resultado final o error
- bibliografía con fuentes oficiales.
- conclusión final y plan de instalación.

## Consejo didáctico

No gana quien nombra más distribuciones.  
Gana quien **justifica mejor** por qué esas 3 sí tienen sentido para este equipo.
