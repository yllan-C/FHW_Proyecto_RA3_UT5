# 20 · Pruebas en máquina virtual

En esta carpeta se documentan las **3 pruebas obligatorias** en VM.

## Qué debe quedar claro

- cómo se ha configurado la máquina virtual;
- qué características del HP real se han intentado simular;
- qué no se puede simular exactamente;
- qué ha ocurrido con cada una de las 3 ISOs;
- qué capturas respaldan la prueba.

## Archivos principales

- `fichas/configuracion_vm.md`
- `fichas/prueba_iso_01.md`
- `fichas/prueba_iso_02.md`
- `fichas/prueba_iso_03.md`

## Capturas mínimas

Por cada ISO, como mínimo:

1. arranque;
2. instalador;
3. resultado final o error.

- `assets/img/20-vm_iso_01/`
- `assets/img/21-vm_iso_02/`
