# 20 · Instalación en el equipo real

En esta carpeta se documenta el proceso de instalación en el **HP Compaq dc7800** usando el USB preparado con Ventoy.

## Archivos de trabajo

- `fichas/registro_general_instalacion.md`
- `fichas/intento_iso_01.md`
- `fichas/intento_iso_02.md`
- `fichas/intento_iso_03.md`

## Requisitos mínimos del bloque

- Debe constar el arranque desde USB.
- Debe documentarse cada intento realizado.
- Debe quedar explicado por qué se continúa o se cambia de ISO.
- Debe quedar demostrado que al final hay **al menos un Linux instalado**.

## Consejo

Piensa en cada intento como una prueba de diagnóstico.  
Aunque falle, no es tiempo perdido: es información útil que debe quedar registrada.
